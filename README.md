# Project Title
Chess board pattern
## Description
This project demonstrates a basic usage of `matplotlib` and `numpy` for visualizing a chessboard pattern.

## Requirements
- Python 3.x
- `matplotlib`==3.8.0
- `numpy`==1.25.0

## Usage
Run the Python script to generate and display the chessboard pattern:
\`\`\`
python main.py
\`\`\`
